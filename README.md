# Ion-Trapping-AFRL-CSRP
This is the repository for the Ion Trapping work done in 2026 by Paige Ramage and Nick Kreidler at AFRL Labs in Rome, NY.

The /Readings folder contains papers to read to learn about or get up to date on quantum technologies so if you have general questions, I would start there.
Each project and test ran is separated into an individual folder. These folders contain the following:
  - Overall Motivation and Explanation
  - Code Used (mainly python)
  - Results (charts, images, qualitative data)

More will be added as we progress through this project.

Disclaimer: ChatGPT and Claude AI were utilized in the development of this project. Previous work from 2024 interns, Gunnar Gott and Erin Clothier, was reused and altered.
